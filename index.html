Headings

Un diseño consistente mantendrá las cosas simples y ordenadas, es por eso que WPKit ui incorpora solo 3 tamaños de headers en porcentajes sobre el tamaño de fuente principal, al 250%, 200% y al 150%. para headings h1, h2 y h3. El resto, h4, h5 y h6 al 100%, esto quiere decir que mantendrán el mismo tamaño que el tamaño de fuente principal, con peso de fuente light (300), regular(400) y bold(700) respectivamente.

Solo en Desktop, Móvil, Tablet y Móvil xs

Las clases wk-d wk-m wk-t y wk-xs muestran contenidos para dispositivos específicos.

	<div class="wk-d">
		<!--Solo aparecerá en desktop (mayor a 771px) -->
	</div>


	<div class="wk-m">
		<!--Aparecerá en cualquier dispositivo móvil (menor a 770px) -->
	</div>


	<div class="wk-t">
		<!--Solo aparecerá en tablet (entre 415px y 770px) -->
	</div>


	<div class="wk-m-xs">
		<!--Solo aparecerá en dispositivos móviles pequeños (menor a 414px) -->
	</div>

Systema de contenedores

Cada porción del layout podrá incluirse en un contenedor con la clase wk-section esta por si sola no aplica ninguna propiedad al contenido pero prepara el mismo para ser contenido en una envolvente con ancho limitado especificanto un contenedor hijo wk-section-wrap. Será necesario definir antes el ancho de este contenedor mediante la clase en body wk-wrap-($width) donde $width = 700px a 1600px siendo posible especificar unicamente medidas pares p.ej. 702, 704, 706... 1598, etc. 

	<div class="wk-section">
		<div class="wk-section-wrap">
			<!--Content-->
		</div>
	</div>


	<!-- wk-wrapp-1100 podrá contener valores 
	de 700 a 1600 con medidas pares únicamente.-->
	<body class="wk-wrap-1100">
		<div class="wk-section">
			<div class="wk-section-wrap">
				<!--Content-->
			</div>
		</div>	
	</body>

Sistema de columnas

Sin más ni más puedes empezar a utilizar el sistema de columnas especificando un contenedor wk-cols y contenedores hijos wk-col. Es muy simple, "cols" plural para el contenedor, "col" singular para cada columna. 

 	<div class="wk-cols">
		<div class="wk-col">Columna 1</div>
		<div class="wk-col">Columna 2</div>
	</div>

Esto hará automáticamente un layout de columnas iguales, no importa cuantas columnas coloques 2 o 100, realmente improbable pero aún así todas las columnas tendrán la misma proporción gracias al poder de Flexbox

¿Preocupado por la compatibilidad con IE9 o navegadores que no soportan Flexbox?, estas cubierto, WPKit ui cargará estílos soportados por esos navegadores de forma automática, o si lo prefieres, puedes anular Flexbox en cualquier explorador; simplemente añade la clase .no-flexbox a la etiqueta html

Cambiar el sistema de columnas

WPKit ui te provee con 4 sistemas de columnas 8, 12, 14 y 16, el default es 12 ya que es el más común así que no será necesario hacer alguna especificación para trabajar con él. Por el contrario para activar el resto de los sistemas solo deberás añadir la clase .wk-cols-8 .wk-cols-14 y .wk-cols-16 respectivamente a un contendor para .wk-cols de la siguiente forma: 

	<div class="wk-cols-8">
		<div class="wk-cols">
			<div class="wk-col-2">Columna 1</div>
			<div class="wk-col-4">Columna 2</div>
			<div class="wk-col-2">Columna 3</div>
		</div>
	</div>

Cambiar el sistema de columnas de forma global

El método anterior activara el sistema de columnas especificado en cada bloque de columnas, lo que puede resultar en una tarea tediosa y complicada si deseas aplicar determinado sistema de columnas a todo el layout.

Para aplicar cualquiera de los sistemas de columnas de forma global solo deberas añadir la clase .wk-cols-8 .wk-cols-14 o .wk-cols-16 a un contenedor global como la etiqueta de body o html. Esto heredará el sistema a todos los bloques de columnas sin importar su ubiación en el layout. 

	<!-- Al aplicar la clase wk-cols-8 a la etiqueta "body", todos los 
	contenedores wk-cols heredan las clases que construyen las columnas-->	
	
	<body class="wk-cols-8">
		<header>
			<h1>Whatever title</h1>
			<section id="whichever-section">
				<!-- Heredará el sistema de columnas de 8 módulos-->
				<div class="wk-cols">
					<div class="wk-col-2">Columna 1</div>
					<div class="wk-col-6">Columna 2</div>
				</div>
			</section>
		</header>
		<!-- Heredará el sistema de columnas de 8 módulos-->
		<div class="wk-cols">
			<div class="wk-col-1">Columna 1</div>
			<div class="wk-col-6">Columna 2</div>
			<div class="wk-col-1">Columna 3</div>
		</div>
	</body>
